# Client dashboard updates — conventions

Short client-visible updates for the password-protected dashboard **Updates** section (`/updates/`).

**Glossary:** [`../README.md`](../README.md)

Set `clientVisible: true` explicitly to publish.

```yaml
---
title: Update Title
summary: One sentence shown in dashboard lists
date: 2026-06-01
category: ops
clientVisible: true
---
```

Use `content/documents/` for formal dashboard documents and `content/updates/` for status notes.
