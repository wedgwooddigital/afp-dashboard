## Design Implications for Chart Redesign

### Client Brief Requirements (from WhatsApp):
- Text under meat types must be **bigger**
- Temperatures should be **bold** (like headings)
- Remove white border, keep black background
- Shorter descriptions (4 short bullets instead of lengthy copy)
- Can flip to portrait for more room if needed
- Must maintain both **clean aesthetics AND educational content**

### Platform Context:
1. **Shopify** needs the chart to feel like a lifestyle accessory, educational tool, confidence-builder
2. **Amazon** needs the chart to be a clear, durable reference tool with professional credibility
3. **Both** emphasize the "Goldilocks" branding and the magic of knowing exact temperatures

### Key Design Opportunities:
- **Readability at distance** - Can it be read from across the kitchen? (Important for both use cases)
- **Professional + Clean aesthetic** - Must balance "clean design" with "instructional content"
- **Meat type hierarchy** - Make it scannable (logo + meat types + temperatures)
- **Temperature prominence** - Bold temperatures align with both platforms' messaging
- **Compact copy** - Shorter descriptive text under each meat type (client's request)
- **Material appearance** - The wipe-clean PVC durability message should be supported by design (doesn't need text, but visual treatment matters)

## Design Direction Notes

The chart redesign should serve **two distinct customer personas simultaneously:**

1. **The Aspiring Home Chef (Shopify-primary):**
   - Wants to learn and improve
   - Values the educational aspect
   - Appreciates clean, professional design
   - Uses chart as teaching tool

2. **The Practical Organizer (Amazon-primary):**
   - Wants quick, reliable reference
   - Values durability and clarity
   - Appreciates no-nonsense design
   - Uses chart as durable kitchen tool

**The solution:** Clean, bold, professional design with **prominent temperatures and clear meat labels**, supporting both confidence-building AND practical reference use cases.
