# Social Media Templates — Layout and Copy

Companion to `design.md` § Social Media Application. **Layout specs for compositing** — copy blocks, dimensions, colours, and logo placement when a designed card is needed. Image models do not render this reliably; see `generation-workflow.md` for when to composite vs caption-only.

## Canvas sizes

| Format | Dimensions | Primary use |
|---|---|---|
| Square | 1080 × 1080 | Instagram feed, Facebook feed |
| Story / Reel | 1080 × 1920 | Stories, Reel covers |
| Landscape | 1200 × 628 | Twitter/X, link previews |
| FB Cover | 820 × 312 | Facebook page cover |

## Template 1: Temperature tip

**Purpose:** Educational content — high engagement for air fryer audience.

```
┌─────────────────────────────┐
│  [Logo — top left, small]   │
│                             │
│  CHICKEN BREAST             │  ← Rubik, Brand Orange or White
│                             │
│  74°C                       │  ← Roboto Mono Bold, White, large
│                             │
│  internal temp · safe min   │  ← Roboto Mono Regular, White/Grey, small
│  · check early in air fryer │
│                             │
│  Dark Navy background       │
└─────────────────────────────┘
```

**Copy formula:** `[PROTEIN]` + `[TARGET TEMP]` + `[one-line context]`

Temperature values: always from `company/temperature-guide.md` (definitive chart: `company/brand/assets/images/graphics/air-fryer-chart-final-preview.png`). Do not use carry-over pull temps.

**Example caption:**
> Chicken breast is done at 74°C internal. In an air fryer, check early — it cooks faster than you think. No more dry chicken. 🌡️

## Typography on social

| Element | Font | Colour | Size (1080 square) |
|---|---|---|---|
| Headline | Rubik Extra Bold (800) or Bold (700) | Brand Orange or White | 72–96px |
| Temperature value | Roboto Mono Bold | White | 96–120px |
| Body / caption on image | Roboto Mono Regular | White | 32–40px |
| Attribution | Roboto Mono Regular | Mid Grey `#666666` | 24–28px |
| CTA button | Roboto Mono Medium | White on Orange | 32px |

**Rule:** Maximum two type sizes per frame (excluding logo).

## Logo placement on social

Use SVG from `company/brand/assets/logos/` — never retype the wordmark.

| Position | Size | Background | Asset |
|---|---|---|---|
| Top left | 120–160px wide | Dark Navy / Charcoal | `artboard-2-on-dark.svg` |
| Top left | 120–160px wide | Brand Orange | `artboard-2-reverse.svg` |
| Bottom right (watermark) | 80–100px wide | Over photo + 20% dark overlay | `artboard-2-on-dark.svg` |
| Centre (cover only) | 200–280px wide | Dark Navy solid | `artboard-2-on-dark.svg` |
| Avatar / profile | 48px+ wide | Dark | `artboard-2-icon.svg` |
| Avatar / profile | 48px+ wide | Orange | `artboard-2-icon-reverse.svg` |

Never place logo on busy photography without overlay.
